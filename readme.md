# TheCollapse

JavaScript / HTML / CSS · Node.js · Docker

---

## 概要

**TheCollapse** は、チャット・履歴管理・アカウント機能・ミニゲームを備えた Web アプリケーションです。  
Node.js をバックエンドとし、Docker による環境構築に対応しています。

---

## 技術スタック

| 分類 | 内容 |
|------|------|
| フロントエンド | HTML / CSS / JavaScript |
| バックエンド | Node.js (index.js) |
| インフラ | Docker |

---

## セットアップ

### Docker を使う場合

```bash
docker build -t thecollapse .
docker run -p 3000:3000 thecollapse
```

### ローカルで直接起動する場合

```bash
npm install
node index.js
```

## 機能一覧

| 機能 | 説明 |
|------|------|
| 💬 チャット | リアルタイム送受信・過去ログ表示・メッセージ削除の即時反映 |
| 🕘 閲覧履歴 | 過去のアクセス履歴を記録・確認 |
| 👤 アカウント | 個人データ管理・今後の同期機能拡張に向けた基盤 |
| 🔄 履歴同期 | 複数端末間での履歴共有 |
| 🎮 ミニゲーム | 軽量なゲームコンテンツ（拡張予定） |

### v1.3

#### 🛡️ 脆弱性修正（セキュリティ強化リリース）
既知の脆弱性を網羅的に洗い出し、可能な限り対処した安全性重視のリリース。

##### ⚠️ 既知の注意点
- `wisp-server-node` は upstream で deprecated 警告が出ているが、後継 `@mercuryworkshop/wisp-js` への移行は API 差分が大きく、プロキシ機能の互換性確認が別途必要なため本リリースでは見送り。
- プロキシ用途のアプリケーションのため、SSRF 的なアクセス自体は機能要件であり制限していません。設置者側でネットワーク的なアクセス制御を行うことを推奨。

#### 🧹 不要データ整理
未使用の重複ファイルや不要データを削除し、プロジェクト全体を軽量化。ディレクトリ構成を整理し管理性を向上。

一般的なホスティング環境での安定動作を目指してサーバー構成を簡略化。

保存処理を最適化して負荷を軽減。ローカルバックアップ保存の追加、ページ終了時の保存漏れ対策、一定間隔での自動保存を実装。
